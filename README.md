# One-Piece-Fansite

**Javascript Features**

* Swipe Up
    * Desc : Menggeser landing page menuju konten dibawahnya (sliding content)
    * How To Interact : Klik tombol dengan icon "jangkar" di bagian bawah landing page

* Dropdown Menu
    * Desc : Memunculkan elemen menu dan menyembunyikannya
    * How To Interact : Klik tombol dengan icon "kemudi" di pojok kiri atas website, pilih menu yang diinginkan, ketika melakukan klik selain elemen tersebut maka ia akan menutup dengan sendirinya

* Looping Navigation Item
    * Desc : Melakukan iterasi untuk menampilkan setiap elemen menu/navigasi
    * How It Work : Menggunakan manipulasi DOM untuk membuat dan meletakkan elemen secara berulang/iterasi  

* Sliding Content
    * Desc : Menampilkan 3 konten bergantian secara otomatis dengan interval 5 detik
    * How It Work : Menggunakan manipulasi DOM untuk menambahkan atau menghapus class "active", class tersebut diatur dalam css dengan mengubah properti display

* Smooth Sliding Gallery
    * Desc : Menampilkan beberapa foto secara otomatis dengan interval 3 detik
    * How It Work : Menggunakan manipulasi DOM untuk menggeser/translasi horizontal setiap elemen fotonya
